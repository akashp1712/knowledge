# Personal Knowledge Base

A personal knowledge base built with [Docus](https://docus.dev) and [Nuxt](https://nuxt.com).

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

## Structure

```
content/              # Markdown content
├── index.md         # Homepage
├── 1.getting-started/  # Getting started section
└── 2.essentials/    # Essentials section
```

## Build

```bash
npm run build
```
