# AI Agents

AI agents are autonomous systems that can perceive their environment, make decisions, and take actions to achieve specific goals. This section covers various agent architectures and patterns.

## Agent Types

- **ReAct Agents** - Reasoning and Acting agents that combine thinking with tool usage
- **Planning Agents** - Agents that break down complex tasks into steps
- **Multi-Agent Systems** - Multiple agents working together
- **Autonomous Agents** - Self-directed agents with minimal human intervention
